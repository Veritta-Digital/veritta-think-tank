# RELEASE NOTES v0.1

## Scope
This release establishes the first disciplined public-facing shape of the Verittà Think Tank repository.

## Included in v0.1
- manifesto and publication policies;
- public method and source-of-truth map;
- glossary and FAQ;
- research agenda and licensing direction;
- V-COF public notes;
- essay series 01–06;
- first public casebook;
- public release readiness note.

## What changed in this release
- the repository now has a clearer entry point for external readers;
- the public/private boundary is explicitly named;
- V-COF public notes were expanded beyond a minimal constitutional stub;
- the essay series was densified to a level more suitable for public reading;
- the first curated casebook was added;
- release discipline and readiness curation were introduced.

## What remains intentionally absent
- raw forge diaries;
- live internal source-of-truth documents;
- sensitive operational details;
- undiscriminated archival publication.

## Release posture
This is not a finished public library.
It is the first coherent public release candidate of the repository.
