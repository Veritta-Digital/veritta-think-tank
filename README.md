# Verittà Think Tank

The public editorial and research repository of the Verittà Think Tank.

This repository exists to make the Verittà body of work legible in public form without confusing the public layer with the sovereign internal canon that governs live operations.

## Mission
Publish serious, auditable and intellectually disciplined work on:
- operational AI governance;
- agentic systems with human sovereignty in the loop;
- fail-closed as an operational philosophy;
- debt governance, evidence and auditability;
- V-COF as a constitutional and operational framework.

## Document placement convention
- foundations are created in [`docs/foundations/`](docs/foundations/README.md)
- essays are created in [`docs/essays/`](docs/essays/README.md)
- Forge Diaries program governance is created in [`docs/forge-diaries/program/`](docs/forge-diaries/program/README.md)
- volume materials are created in [`docs/forge-diaries/volume-public-0.1/`](docs/forge-diaries/volume-public-0.1/README.md)
- outward channel material is created in [`docs/outward/linkedin/`](docs/outward/linkedin/README.md)

The repository now includes a structural guardrail in [`scripts/check-docs-structure.ps1`](scripts/check-docs-structure.ps1) to block new loose `.md` files in `docs/` root, except for `docs/index.md`.

## Public-layer principles
- nothing sensitive is published without curation and redaction;
- raw forge diaries are never published directly;
- public material should remain intelligible to qualified external readers;
- the public layer must not pretend to replace the sovereign internal canon of the Verittà ecosystem.

## Current public architecture
This repository currently contains:
- manifesto and public policies;
- public-facing V-COF notes;
- glossary and FAQ for external readers;
- research agenda;
- essay series;
- redacted casebooks and publishing templates;
- release discipline and publication readiness notes.
