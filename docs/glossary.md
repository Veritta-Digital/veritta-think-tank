# Public Glossary

This glossary is a public-facing, simplified layer aligned with the canonical V-COF glossary in the internal HQ.
It does not replace the sovereign internal glossary.

## V-COF
V-COF is the Verittà Cognitive-Operational Framework.
Its procedural core operates through contract-governed operations, but the official public name of the framework remains singular and stable.

## FAIL-CLOSED
The default operational posture of V-COF.
When a critical ambiguity, missing precondition or blocking uncertainty appears, the system stops and declares the problem explicitly instead of improvising silently.

## GATE
An explicit checkpoint that must be cleared before the process advances.
In governance terms, gates structure approval, review and closure.

## SEAL
The evidence bundle that documents the verifiable completion of an operation.
A seal is not decoration. It is structured proof that something happened under governed conditions.

## SoT (Source of Truth)
The canonical document designated to govern a specific class of information.
The SoT is not merely the newest file. It is the file with formal precedence.

## AGC
An Agentic Governance Capsule.
It is the bounded envelope around an agent call: scope, constraints, expected output and failure conditions are declared so the agent does not operate in free expansion mode.

## EXECUTION NOTE
A human-readable factual record of a technical operation.
It complements the seal rather than replacing it.

## COUPLING
The process of connecting a new workspace, session or agent instance to the relevant operational context.
Without proper coupling, terminology drift and duplicated work become likely.

## REENTRY
The session-return protocol used to reconstitute enough context for safe work.
It operationalizes coupling at the level of a new or resumed session.

## JURIMETRIC CODING
The epistemic discipline that treats code, documentation and execution with the same demand for proof, traceability and procedural rigor that a legal process demands.

## TOMBSTONE
A terminal custody state for material that is dead, migrated, memorial or superseded, while preserving trail and minimum human navigability.

## SUPERSEDENCE
The formal condition in which one artifact has been explicitly replaced by a more current, more precise or more canonical successor.

## MEMORIA DIGNIFICADA
The discipline of preserving operationally or historically valuable context without collapsing the workspace into indiscriminate accumulation.
