# Debt Governance for Agentic Systems

Dívida em sistemas agentic não é apenas técnica.
Ela pode ser cognitiva, documental, operacional, de avaliação ou de governança.

Este texto abrirá a discussão sobre:
- debt registries;
- priorização baseada em risco;
- documentação como fonte de verdade;
- diferença entre sistema funcional e sistema governado.
