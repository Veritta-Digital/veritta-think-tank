# LinkedIn Launch Post — PT-BR (Final)

Hoje tornamos pública a primeira edição governada dos **Diários da Forja**.

**Forge Diaries — Public Volume 0.1** não é um arquivo bruto.
É uma edição curada sobre:
- governança de IA agentic,
- continuidade entre sessões, sistemas e decisões,
- disciplina adversarial,
- e a passagem entre rigor operacional privado e voz pública.

O volume segue uma regra central:
**RAW != PUBLICÁVEL**

Seu objetivo não é revelar tudo.
Seu objetivo é tornar a arquitetura da forja legível em forma pública, sem confundir publicação com exposição indiscriminada.

Esta primeira edição se organiza em quatro partes:
1. **O Prólogo Constitucional**
2. **O Crisol Adversarial**
3. **Governança, Memória e Forma**
4. **O Limiar para a Linguagem Pública**

Ao lado do volume, dois essays satélites já foram promovidos na série pública:
- **Why Serious AI Governance Must Capture Decision Rationale**
- **Governance Does Not Eliminate Freedom**

No fundo, este projeto parte de uma convicção simples:
**governança de IA não é linguagem decorativa de policy — é arquitetura operacional.**

Se quiser acompanhar a evolução pública desta linha, este é apenas o começo.
