# Licensing Direction

## Current position
The Verittà Think Tank can publish public concepts, essays and framework notes openly without immediately exposing the full operational kit behind them.

## Strategic direction
The most coherent long-term model is likely:

- **public open layer** for concepts, essays, explanatory framework notes and selected educational material;
- **private licensed layer** for operational playbooks, implementation kits, templates, advanced governance tools and adoption artifacts.

## Why not publish everything?
Because the internal canon contains:
- sensitive operational specifics;
- proprietary implementation logic;
- client-adjacent knowledge;
- governance material that should be translated before publication.

## Why not keep everything closed?
Because public intelligibility matters.
A framework that cannot explain itself publicly struggles to become legible, discussable and eventually licensable.

## Recommended model
**Open-core for thought. Licensed-core for operation.**

## Licensing caution
The licensing layer should only be formalized after:
- public terminology is stable;
- the public/private split is well defined;
- the kit to be licensed is coherent and repeatable;
- casebooks and examples are safely redacted.

## Immediate implication
This repository can continue publishing public framework material now, while preparing a future private licensing kit without collapsing the distinction between the two.
