# Casebooks

Esta pasta será usada para estudos de caso públicos, sempre em formato redigido, anonimizado e curado.

## Regras
- nenhum casebook nasce bruto;
- nenhum casebook expõe cliente, tenant ou incidente identificável;
- cada casebook deve explicar contexto, problema, intervenção, limites e aprendizado;
- sempre que necessário, separar fato, inferência e hipótese.
