# Casebooks

This folder contains public case studies in redacted, anonymized and curated form.

## Rules
- no casebook begins as a raw dump;
- no casebook exposes clients, tenants or identifiable incidents;
- each casebook should explain context, problem, intervention, limits and lessons;
- when relevant, fact, inference and hypothesis should remain distinct.
