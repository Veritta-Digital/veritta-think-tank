# LinkedIn Package

This folder contains the outward publishing package for **Forge Diaries — Public Volume 0.1** with a specific focus on LinkedIn.

## Purpose
Prepare the volume for public-facing LinkedIn publication in a way that preserves Verittà Premium editorial discipline while remaining native to LinkedIn's publishing surfaces.

## Package contents
- `LINKEDIN_PUBLICATION_STRATEGY.md`
- `LINKEDIN_LAUNCH_POST_EN.md`
- `LINKEDIN_LAUNCH_POST_PT_BR.md`
- `LINKEDIN_NEWSLETTER_CONCEPT_EN.md`
- `LINKEDIN_NEWSLETTER_CONCEPT_PT_BR.md`
- `PORTUGUESE_TRANSLATION_POSITION.md`

## Editorial logic
The volume remains the master editorial object.
LinkedIn materials are outward-facing derivatives optimized for discovery, circulation and subscriber capture.
