# Bilingual Public Layer Note

## Purpose
This note records the editorial logic of maintaining both an English public layer and a Portuguese public layer for the Forge Diaries project.

## Why bilingual
The English layer supports international legibility and future outward positioning.
The Portuguese layer supports first-wave reach, clarity and stronger resonance in the Brazilian context.

## Editorial rule
The two layers should remain structurally faithful to one another, but they do not need to be mechanically identical.

Portuguese may adapt rhythm, framing and public-facing emphasis to local intelligibility, while preserving the same governing architecture.
