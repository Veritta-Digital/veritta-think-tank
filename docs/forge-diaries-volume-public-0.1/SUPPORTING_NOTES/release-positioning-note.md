# Release Positioning Note

This volume should be positioned as a **first governed public edition**.
It should not be described as the totality of the Forge Diaries corpus, nor as a raw documentary archive.

Its public posture is better captured by three terms:
- curated;
- admissible;
- structurally faithful.

The purpose of this release is not exhaustive disclosure.
Its purpose is to make the architecture of the forge legible in public form.
