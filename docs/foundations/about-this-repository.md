# About This Repository

## What this repository is
This is the public-facing editorial and research repository of the Verittà Think Tank.
It exists to publish concepts, essays, framework notes and redacted case material derived from a deeper private operational canon.

## What this repository is not
This repository is not the sovereign operational HQ of the Verittà ecosystem.
It does not contain raw diaries, operational secrets, live contracts, sensitive topology or the full internal constitutional apparatus used in production governance.

## Why this distinction matters
Public trust is not built by publishing everything indiscriminately.
It is built by publishing material that is:
- true to the deeper canon;
- legible to external readers;
- bounded by redaction;
- stable enough to carry public meaning.

## Audience
This repository is intended for:
- researchers and operators interested in AI governance;
- readers studying agentic systems and human-in-the-loop design;
- collaborators evaluating the conceptual maturity of the Verittà body of work;
- future partners, clients or institutions seeking to understand the public-facing layer of V-COF.

## Working principle
The public layer should explain without overexposing.
It should clarify without pretending totality.
It should make the framework discussable without violating the integrity of the private canon from which it was derived.
