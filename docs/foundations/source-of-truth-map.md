# Source of Truth Map

## Principle
A public repository can publish truthfully without pretending to be the only or ultimate source of truth.

In the Verittà ecosystem, **sovereign operational source-of-truth documents remain internal**.
This public repository is a curated, publishable, redacted layer derived from that deeper canon.

## Internal sovereign layer
The internal HQ governs:
- constitutional rules
- operational procedures
- agent hierarchy
- roadmap and sequencing
- project governance
- seals, contracts and execution notes

## Public layer
This repository governs:
- public essays
- public framework explanations
- externally readable glossary entries
- public-facing research agenda
- redacted casebooks and educational material

## Rule of precedence
If a public document here diverges from the sovereign internal canon, the internal canonical source prevails.
The public layer must then be corrected.

## Editorial consequence
This repository should never claim:
- operational completeness;
- visibility over all private artifacts;
- authority over internal sequencing;
- authority over raw forge material.

It should claim only what it can publicly sustain.
