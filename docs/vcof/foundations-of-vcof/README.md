# Foundations of V-COF

This folder contains the first public companion series derived from the V-COF public materials.

The purpose of the series is didactic and institutional.
It does not replace the constitutional note, the governance principles or the Forge Diaries volume.
It offers a short, sequential path for readers who need to understand the framework through a guided public reading order.
