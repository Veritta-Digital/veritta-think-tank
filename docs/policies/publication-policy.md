# Publication Policy

## Regra-mãe
RAW != PUBLICÁVEL

## Camadas
1. Privado canônico
2. Curadoria intermediária
3. Público editorial

## Critérios mínimos para publicação
- Redaction concluída
- Revisão de sensibilidade concluída
- Escopo definido
- Evidência suficiente para os claims centrais
- Aprovação humana final

## Não publicar
- segredos
- topologia interna sensível
- incidentes identificáveis
- dados de clientes/tenants
- diários brutos integrais
